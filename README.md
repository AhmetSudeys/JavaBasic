# ☕ Java Journey: From Basics to Games

Bu depo, Java programlama diline sıfırdan başlayıp kendimi geliştirdiğim tüm süreci kapsamaktadır. Temel sözdiziminden (syntax) grafiksel oyun geliştirmeye kadar uzanan geniş bir yelpazeyi içerir.

---

## 📂 Klasör Yapısı ve İçerik

### 🌱 [BaslangicProjelerim](./BaslangicProjelerim)
Java programlamanın yapı taşlarını ve algoritma mantığını tamamen kavradığım, dilin "mutfağına" dair her şeyi barındıran kapsamlı bir temel rehberdir.

### 🏗️ [GenelProjeler](./GenelProjeler)
Öğrendiklerimi bir üst seviyeye taşıdığım orta düzey konular:
* **OOP (Nesne Yönelimli Programlama):** Sınıflar, miras alma ve polimorfizm.
* **Hata Yönetimi:** Try-catch blokları ile güvenli kodlama.
* **Grafik & Çizim:** Java Swing/AWT ile görselleştirme ve grafik çalışmaları.
* **Dosya İşlemleri:** Veri okuma ve yazma pratikleri.
* **Listeners:** Kullanıcı etkileşimleri ve olay yönetimi.

### 🎮 [Oyunlar](./Oyunlar)
Öğrenme sürecinin en keyifli kısmı! Bu klasörde Java ile geliştirdiğim **6 farklı oyun** bulunmaktadır. Her biri hem mantıksal hem de görsel olarak farklı bir meydan okumayı temsil ediyor:
* Birbirinden eğlenceli ve öğretici 6 farklı masaüstü oyunu.

---

## 🚀 Nasıl Çalıştırılır?
Bu projeler standart Java SE kullanılarak geliştirilmiştir.
1. Bilgisayarınızda bir JDK (Java Development Kit) kurulu olduğundan emin olun.
2. Repoyu klonlayın: `git clone https://github.com/AhmetSudeys/JavaBasic.git`
3. İstediğiniz klasöre giderek ilgili `.java` dosyasını IDE'niz (IntelliJ, Eclipse, VS Code vb.) üzerinden çalıştırın.
